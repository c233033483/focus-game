# Solas

Solas is a cozy cafe sim developed in Unity 6 as part of a serious game collaboration with Focus Ireland. Across five days of gameplay, players serve customers, listen carefully to their orders, build trust, and help those in need.

Solas was produced by a team of five for our serious game module in college. The game runs in the brower using WebGL and supports both mouse and touch input.

Play it here: https://sebkii.itch.io/solas
